## Prerequisites

What things you need to install the software and how to install them

- [Node](https://nodejs.org) v12.16.3 or higher
- [Yarn](https://yarnpkg.com) v1.22.4 or higher

## Installation

```bash
yarn install
```

## Running the app

```bash
yarn dev
```

## Test

```bash
# unit tests
yarn test

# code style
yarn validate
```
